Лабораторная работа №6. Алгоритм Axо-Корасик.

Алгоритм сборки:
1) Перейти в папку lr6, прописав cd lr6
2) Прописать sudo chmod +x build.sh
3) Затем прописать ./build.sh
4) Для запуска программ перейти в папку build, прописав cd build

Алгоритм Ахо-Корасик:
1) Находясь в папке build написать ./AKStandart
2) Ввести текст
3) Ввести число шаблонов
4) Ввести n шаблонов
5) Вывод - все вхождения шаблонов в текст

Алгоритм Ахо-Корасик с Джокерами:
1) Находясь в папке build написать ./AKDjoker
2) Ввести текст
3) Ввести шаблон
4) Ввести символ джокера
5) Вывод - строки с номерами позиций вхождений шаблона

Вызов (проверка) тестов:
1) Находясь в папке build написать ./tests

Для очистки:
1) Находясь в папке build написать ./clean
