Лабораторная работа №5. Алгоритм КМП.

Алгоритм сборки:
1) Перейти в папку build
2) Прописать cmake ..
3) Затем прописать make

Алгоритм КМП:
1) Находясь в папке build написать ./kmp
2) Ввести шаблон (Template)
3) Ввести текст (Text)
4) Вывод - индексы начал вхождений шаблона в текст в случае успеха, иначе -1

Проверка на циклический сдвиг:
1) Находясь в папке build написать ./shift
2) Ввести первую строку (str1)
3) Ввести вторую строку (str2)
4) Вывод - индекс начала второй строки в первой в случае успеха, иначе -1

Вызов (проверка) тестов:
1) Находясь в папке build написать ./test

Для сравнения эффективности алгоритмов КМП и наивного по кол-ву операций:
1) Находясь в папке build написать ./NaiveVsKmp

Для одновременного вызова КМП, циклического сдвига и тестов:
1) Находясь в папке build написать ./all

Для очистки:
1) Находясь в папке build написать ./clean
